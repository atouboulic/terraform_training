# TP 01 - Installation Terraform et projet simple


## Prérequis

Installer terraform : https://developer.hashicorp.com/terraform/tutorials/aws-get-started/install-cli#install-terraform

Installer IntelliJ : https://www.jetbrains.com/fr-fr/idea/ (version commununautaire)
+ Installer le plugin "Terraform and HCL" (File > Settings > Plugins)

(OU Installer VS CODE : https://code.visualstudio.com/ 
+ Installer l'extension Hashicorp Terraform : https://marketplace.visualstudio.com/items?itemName=HashiCorp.terraform

Créer un dossier nommé tp_terraform et un dossier TP01 dans le dossier tp_terraform.

## Objectif du tp

Faire un projet terraform pour la première fois. Il faudra
+ chercher un provider sur le registry
+ l'installer
+ le configurer

Puis il faudra lancer les commandes terraform appliquer ce projet.

## Mon premier projet Terraform

Dans le dossier TP01, créer 2 fichiers : 
   - main.tf
   - providers.tf

Chercher le provider 'random' d'hashicorp sur le registry provider de Terraform.

Récupérer les infos du provider et mettre à jour le fichier providers.tf

Identifier la resource qui permet de créer une string aléatoire.

A partir de cette resource et de la documentation, créer le code qui va 
+ générer la resource "string aléatoire sur 12 caractères sans caractères spéciaux

Lorsque le code est prêt, faire la commandes suivantes :

```shell
terraform init
```

Observer les fichiers/répertoires créés dans le répertoire TP01

Puis faire cette commande :

```shell
terraform plan
```

Observer qu'il n'y a pas de changement dans le répertoire

Enfin faire cette commande :

```shell
terraform apply
```

Bravo ! Vous venez de créer une resource avec terraform !

Maintenant observer le nouveau fichier créé dans le répertoire suite à cette commande.
